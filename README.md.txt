## Deskripsi Proyek

Aplikasi Blog merupakan aplikasi berbasis web yang digunakan untuk mengelola artikel blog. Pengguna dapat menambahkan, mengubah, menghapus, dan melihat artikel yang tersimpan dalam database.

## Fitur

* Login Admin
* Manajemen Artikel (CRUD)
* Upload Gambar Artikel
* Kategori Artikel
* Pencarian Artikel
* Dashboard Admin

## Teknologi yang Digunakan

* PHP
* MySQL
* HTML
* CSS
* Bootstrap
* JavaScript

## Cara Menjalankan Proyek

1. Clone repository:

   ```bash
   git clone https://github.com/username/aplikasi-blog-123456.git
   ```

2. Pindahkan project ke folder `htdocs` (XAMPP).

3. Buat database MySQL.

4. Import file database:

   ```
   database.sql
   ```

5. Sesuaikan konfigurasi database pada file:

   ```
   .env
   ```

   atau

   ```
   koneksi.php
   ```

6. Jalankan Apache dan MySQL melalui XAMPP.

7. Buka browser:

   ```
   http://localhost/aplikasi-blog
   ```
